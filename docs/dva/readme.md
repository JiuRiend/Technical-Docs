# Dva

## 预备知识

- es6
- generator 
- redux-sage
- webpack

## 安装

```cmd
    npm install -g dva-cli 
    dva -v
```

## 初始化
```
    dva new <projectname>
```

## react项目集成dva

```
    npm install -g create-react-app
    create-react-app <projectname>
    cd <projectname>
    yarn install
    yarn run eject 

    yarn add dva --save
```
